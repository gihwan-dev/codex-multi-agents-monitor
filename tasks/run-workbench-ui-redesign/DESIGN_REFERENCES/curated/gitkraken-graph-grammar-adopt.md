# Adopt Reference

- Title: GitKraken Commit Graph
- Source URL: https://www.gitkraken.com/features/commit-graph
- Why selected:
  - relation-first graph 문법이 강하다.
  - left navigation -> primary graph -> detail inspection 흐름이 분명하다.
  - 세로 읽기 순서가 명확해 dense한 정보도 방향을 잃지 않는다.
- Use in this task:
  - row-based graph의 reading order
  - sticky lane header와 explicit relation emphasis
  - selected path를 카드 강조가 아니라 관계 강조로 보여주는 방식
- Do not copy:
  - git-specific naming, commit-centric semantics
