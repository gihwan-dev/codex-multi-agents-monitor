# Adopt: GitKraken Graph Grammar

- Source: https://www.gitkraken.com/features/commit-graph
- Use for: lane/edge grammar, three-pane layout hierarchy, contextual graph action
- Adopt:
  - 세로 흐름 graph를 compact metadata column과 결합하는 방식
  - left navigation, central graph, right detail panel의 명확한 역할 분리
  - graph 위에서 바로 action/context를 여는 interaction mindset
- Do not copy:
  - Git-specific author/SHA/branch vocabulary
  - commit table density를 그대로 가져오는 것
